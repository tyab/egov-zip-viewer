# eGov ZIP Viewer

A browser-based viewer for Japanese government and administrative ZIP packages.

## Features

- XML + XSL/XSLT rendering
- PDF viewing
- TIFF/JPEG/PNG image viewing
- Shift-JIS / CP932 support
- ZIP archive browsing
- Local-only processing (no file uploads)

## Privacy

All files are processed locally in your browser.
No uploaded files are sent to any server.

## Disclaimer

Rendered output may differ from official government systems and printed forms.
Always verify important information using official documents and services.
